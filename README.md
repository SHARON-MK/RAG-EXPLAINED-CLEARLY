# RAG Explained Clearly

This repository is designed to help you understand Retrieval Augmented Generation (RAG) in a simple and fundamental way — without unnecessary complexity or jargon.

The goal is to focus on the core working mechanism of RAG so you truly understand how it works internally. Once the fundamentals are clear, you can build anything on top of it.

Setup instructions are available in `SETUP.md`.

The codebase includes clear comments wherever needed to guide your understanding step by step.
